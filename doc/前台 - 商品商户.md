## 获取更多商户列表

* url: /shop
* method: GET
* request:
```
{
    id: id值
}

```
* 说明:id值每次传入的值为当前列表的最后一个值。

* response:

```
{
    {
        "msg": "获取成功",
        "data": {
            "shops": [
                {
                    "id": 18,
                    "name": "yaomin",
                    "manager": "yaomin",
                    "tel": "yaomin",
                    "address": "yaomin",
                    "logo": "yaomin",
                    "tag": "",
                    "create_time": 1478334054,
                    "deleted_at": null
                },
                ...
            ]
        }
    }
}

```
