# 商户

## 新增页面
url: 'admin/shops/create'
method: GET

## 新增
url: 'admin/shops'
method: POST
request: 
```
{
    tel: 手机号码,string
    address: 地址,string
    name: 店名,string
    manager: 管理人员,string
    logo: 门店logo,string
}

```

reponse: 店铺信息

```
{
    id: 店铺ID,int
    tel: 手机号码,string
    address: 地址,string
    name: 店名,string
    manager: 管理人员,string
    logo: 门店logo,string
    create_time: 创建时间,int
}
```
